# Design Principles

## 1. Color is structure
Color is not an afterthought. It sets hierarchy, emotion, and flow.

## 2. One scene at a time
Each section should feel intentional and self-contained.

## 3. Contrast beats complexity
A bold flat background with strong text contrast usually works better than overdesigned gradients.

## 4. Limit the palette, increase the confidence
A small bold palette feels more branded than a huge random one.

## 5. CTA clarity wins
If the button does not immediately stand out, the color system is failing.

## 6. UI and artwork must belong together
Mascots, illustrations, and interface colors should feel like they came from the same brand world.

## 7. Loud is fine. Messy is not.
High energy is not the same thing as chaos.

## 8. Mood shifts matter
Section-to-section color changes help the user feel progression.

## 9. White is a design asset
Soft white helps loud colors breathe.

## 10. Use intensity with purpose
Pink is for emotion, indigo is for structure, sky blue is for openness.
