# AI Prompt Library

## Master redesign prompt
Use this when redesigning an existing site.

```text
Redesign this website using a bold, high-energy, mascot-friendly color system. The site should feel alive, memorable, playful, and internet-native without becoming chaotic. Use large flat color fields, strong contrast, rounded CTA buttons, section-based mood shifts, and a tight palette built around sky blue, electric indigo, hot pink, soft white, and one or two rotating scene colors. Keep one dominant background color per section, preserve readability, and make the interface feel like it belongs in the same world as the illustrations and graphics.
```

## CTA upgrade prompt
```text
Improve the CTA system so the primary action is instantly obvious. Use a stronger saturated button color, cleaner contrast, and clearer visual hierarchy. Secondary actions should remain visible but should never compete with the primary CTA.
```

## Section mood prompt
```text
Redesign each section like a scene with its own emotional tone while keeping the overall palette consistent. Use color shifts between sections to create momentum and variety.
```

## Color discipline prompt
```text
Tighten the palette. Remove unnecessary colors, preserve only the strongest brand tones, and make each section rely on one dominant background color, one text color, one accent color, and one CTA color.
```

## Brand-world cohesion prompt
```text
Make the interface, artwork, icons, and headline styling feel like they come from the same brand universe. Do not let the illustrations feel disconnected from the UI palette.
```

## Premium-playful prompt
```text
Keep the site colorful and energetic, but elevate it so it feels designed, intentional, and premium-playful rather than childish or random.
```
